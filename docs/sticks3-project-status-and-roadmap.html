StickS3 Mac Agent Companion：当前成果与拓展路线

这个项目把 Mac 上的 agent 状态、审批请求、系统负载、专注信号和桌面宠物，压缩到一个可以常亮摆在桌面的 StickS3 小屏里。

Screen Home、Agent、Schedule、Power、Pet 和详情页。
Live Link Mac 本地同步 agent、审批、CPU、RAM 和日程。
Buttons A / B 双键承担翻页、详情、返回、审批和语音输入。
Pet 无操作进入宠物待机，忙碌、完成和等待用情绪化状态表达。
Interactive HTML 打开全量模拟器

02 Expansion Map

结合整个项目可以拓展的内容

这些方向都能复用当前的屏幕、按键、Mac 本地同步和实时状态基础。优先级应围绕“桌面上能一眼看懂并能立即操作”来排序。

Agent Approval Buddy

把 Codex、Claude、Cursor 或本地 agent 的权限请求统一到硬件屏。A 批准，B 拒绝，详情页显示风险、命令和来源。

priority 1security

Focus + Interruptibility

接入日历、专注模式和 agent awaiting 状态。屏幕显示是否可以打断，以及哪个 agent 正在等你。

focuscalendar

Mac Ops Mini Monitor

扩展 CPU、RAM、磁盘、网络、温度、风扇和 bridge heartbeat。StickS3 只显示关键告警，大量图表留给 Mac 页面。

low riskops

Personal Now Terminal

把 Calendar、Reminders、Mail、Messages、天气或 RSS 经 Mac bridge 脱敏摘要后送到屏幕，形成桌面即时信息终端。

digestprivacy

Agent Macro Remote

把 StickS3 升级成小型 Stream Deck：停止任务、重试、打开日志、切换 agent、打开语音输入、控制音乐或触发 IDE 快捷动作。

hidremote

Multi Agent Hub

把 Codex、Claude、Cursor、本地脚本或 CI agent 统一成列表，显示 idle、busy、done、error、stale、offline。

schemaagents

Pet System

扩充 mood 动画、睡眠/唤醒、角色切换和工作状态映射。短期重点是把 pet 做成可关闭的 idle 层，而不是单独的装饰页面。

assetsmood

Config / OTA Portal

做一个本地配置页：亮度、默认页面、agent 源、pet 资源、刷新频率、BLE/USB 通道和固件更新状态。

configota

Hardware Variants

StickS3 继续做最小可用状态屏。若要图表、触摸和多窗口，可评估 T-Display-S3、CYD 或 4.3 寸 ESP32-S3 屏。

stick firstlarger screen

当前边界

  • 屏幕StickS3 屏幕小，适合主状态、短标签、单个操作，不适合完整 dashboard。
  • 传输BLE 已跑通，但安全审批链路应单独加固；USB serial 仍是稳定 MVP 备选。
  • 展示HTML 模拟器是演示层；硬件表现以固件 build / flash 后结果为准。
